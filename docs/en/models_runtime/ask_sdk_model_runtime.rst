Models Runtime :ask\_sdk\_model\_runtime package
================================

The Models Runtime classes contains the common models package by the 
Software Development Kit (SDK) team for Python that provides common 
implementations like Serializer, ApiClient BaseServiceClient, 
Authentication Configuration etc. that is used by the SDK models.

.. note::

    Canonical imports have been added in the ``__init__.py`` of the package.
    This helps in importing the class directly from the package, than
    through the module.

    For eg: if ``package a`` has ``module b`` with
    ``class C``, you can do ``from a import C`` instead of
    ``from a.b import C``.
